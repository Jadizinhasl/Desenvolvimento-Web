# RESPOSTAS
1. Como o próprio nome diz, é um protocolo de transferência Hipertexto. Ele serve para a transmissão dos dados pela internet.

2. Os principais metódos HTTP:
- GET
- POST
- PUT
- DELETE

3. A diferença é que um,é mais seguro que o outro, por ter uma camada extra com criptografia dos dados transmitidos pelos protocolos SSL/TLS.

4. É uma arquitetura de aplicação distribuida, dividida entre uma solicitação e um processamento.

5. A diferença é que no GET podemos apenas receber a informação, já no POST podemos adicionar informação de acordo com o que foi recebido.

6. Servidor ou serve, é o "local" que processa os dados salvos e envia, no caso de um restaurante seria a cozinha, onde se pede um prato e o mesmo é devolvido.

7. Datacenter nada mais que um local de forma fisica, que é usado para guardar dados, tais como os servidores.

8. Cliente é quem faz a solicitação, ou seja, um computador, tablet e por ai vai. No mesmo caso do restaurante, seria eu seria a cliente e pediria uma refeição. Cliente é quem solicita.
